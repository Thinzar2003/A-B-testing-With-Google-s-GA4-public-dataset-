02 — Experiment Design

Built to detect
a 10% lift.

The experiment was sized using power analysis with Bonferroni correction for multiple comparisons, and ran for a full business cycle to avoid day-of-week bias.

Null hypothesis H₀

Adding social proof or incentive copy to the CTA button will have no effect on add-to-cart conversion rate.

Alternative hypothesis H₁

Adding "free shipping" or urgency copy will increase add-to-cart CVR by ≥10% relative to control.

Primary metric

Add-to-cart rate = sessions with add_to_cart event ÷ total sessions. Tracked at session level, not user level.

Sample size rationale

At baseline CVR 3.24%, MDE 10%, α=0.05, power=80%, with Bonferroni correction: ~27,800 sessions per variant required.

05 — Implementation

Real data,
reproducible analysis.

Built entirely in Google Colab using the public GA4 BigQuery dataset. Variant assignment is deterministic via FARM_FINGERPRINT hash — no experiment infrastructure required.

-- BigQuery: pull session-level experiment data WITH sessions AS ( SELECT user_pseudo_id, -- deterministic 3-way split via hash CASE WHEN MOD(ABS(FARM_FINGERPRINT(user_pseudo_id)), 3) = 0 THEN 'control' WHEN MOD(ABS(FARM_FINGERPRINT(user_pseudo_id)), 3) = 1 THEN 'variant_b' ELSE 'variant_c' END AS variant FROM `bigquery-public-data.ga4_obfuscated_sample_ecommerce.events_*` WHERE _TABLE_SUFFIX BETWEEN '20210101' AND '20210131' )
# Python: chi-square significance test def ab_test(df, control, treatment): ctrl = df[df.variant == control]['did_add_to_cart'] trt = df[df.variant == treatment]['did_add_to_cart'] contingency = [[int(ctrl.sum()), len(ctrl) - int(ctrl.sum())], [int(trt.sum()), len(trt) - int(trt.sum())]] chi2, p_value, _, _ = stats.chi2_contingency(contingency) uplift = (trt.mean() - ctrl.mean()) / ctrl.mean() return {'uplift': uplift, 'p_value': p_value, 'significant': p_value < 0.05}
